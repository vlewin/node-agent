# Usage
<!-- usage -->
```sh-session
$ npm install -g mynewcli
$ mynewcli COMMAND
running command...
$ mynewcli (-v|--version|version)
mynewcli/0.0.0 darwin-x64 node-v12.12.0
$ mynewcli --help [COMMAND]
USAGE
  $ mynewcli COMMAND
...
```
